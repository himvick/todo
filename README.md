## Todo App

A simple todo app built with create-react-app and typescript

### Steps to Install and Run
- Clone Repository into An empty folder
  - Open your terminal
  - Run "git clone https://github.com/himvick/todo" in an empty folder
- Install Dependencies
  - Navigate to the project's root folder
  - Run "npm install"
  - This will install all the project's dependencies
- Run App
  - Whiles in the root folder Run "npm start"
  - Enjoy
